Persiapan Folder Kerja

Setelah instalasi, siapkan struktur folder untuk data processing:

  1. Buat folder utama untuk project (contoh: D:\STS_KPI_Data\)
  2. Buat sub-folder berikut:
    • Input: untuk file raw data yang akan diproses
    • Output: untuk hasil pemrosesan
    • Klasifikasi: untuk file hasil klasifikasi
    • LOG: untuk file log aplikasi (otomatis dibuat)
  3. Pastikan folder memiliki permission write access

‚ùì Aplikasi tidak bisa start atau crash saat dibuka

üîê Login gagal atau username/password ditolak

üìä Data tidak ter-upload ke database

  • Verifikasi struktur tabel database
  • Monitor log untuk error message detail

  • Clear temporary files secara berkala
